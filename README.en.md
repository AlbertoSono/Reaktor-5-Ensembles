Welcome to this extensive archive of **Native Instruments Reaktor 5 Ensembles**. This repository serves as a long-term preservation project for the creative output of the Reaktor community.

## üöÄ Content

- **Over 400 Files:** A massive collection (approx. 6 GB) of synthesizers, effects, and sequencers.
- **Formats:** Includes `.ens` files, as well as `.zip` and `.rar` archives containing snapshots and modules.
- **Source:** Mostly curated from the legendary [NI Reaktor User Library](https://www.native-instruments.com).

## üì¶ How to Use

1. **System Requirements:** You need **NI Reaktor 5** or higher.
2. **Git LFS (Important):** This repository uses **Git Large File Storage**. If you clone it via command line, ensure you have [Git LFS](https://git-lfs.github.com) installed.
3. **Manual Download:** If you are not familiar with Git, the best way to get the full 6 GB is using [GitHub Desktop](https://desktop.github.com).
