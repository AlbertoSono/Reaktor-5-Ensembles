# 🎹 Reaktor 5 アンサンブル・コレクション
[Deutsch](README.md) | [English](README.en.md) | [Français](README.fr.md) | [Español](README.es.md) | [Português](README.pt.md) | [Italiano](README.it.md) | [日本語](README.ja.md)

**Native Instruments Reaktor 5 アンサンブル** の膨大なアーカイブへようこそ。このリポジトリは、Reaktorコミュニティの資産を長期保存するためのプロジェクトです。

## 🚀 内容
- 400以上のファイル（約6GB）のシンセサイザー、エフェクト、シーケンサー。
- `.ens` ファイルおよび、スナップショット等を含む `.zip/.rar` アーカイブ。

## 📦 使用方法
- **必須環境:** NI Reaktor 5 以降。
- **重要:** このリポジトリは **Git LFS** を使用しています。6GBすべてを正しくダウンロードするには [GitHub Desktop](https://desktop.github.com) の使用を推奨します。

## ⚖️ 免責事項
これは**非営利の保存アーカイブ**です。すべての著作権は、[NI User Library](https://www.native-instruments.com) のオリジナルクリエイターに帰属します。
